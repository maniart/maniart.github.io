Coding time.

In this section, we will go over a number of simple code examples. There are some pre-written pseudo code examples that we will turn into actual code/drawings

one

  1. create a canvas with the dimensions 800 x 600
  2. the canvas background is a light grey
  3. put the pen at coordinates 0,0
  4. set your initial (x,y) to (0,0)
  5. draw a 4px thick black line from coordinates (x,y) to (width,y)
  6. move the pen 20 pixels down along the y axis
  7. repeat steps 5 to 6 until (y) is equal to (height)

two

  1. create a canvas with the dimensions 800 x 600
  2. the canvas background is a light grey
  3. put the pen at coordinates 0,0
  4. set your initial (x,y) to (0,0)
  5. draw a 4px thick black line from coordinates (x,y) to (width,y)
  6. move the pen 20 pixels down along the y axis
  7. repeat steps 5 to 6 until (y) is equal to (height)
  8. set your initial (x,y) to (0,0)
  9. draw a 4px blue line from coordinates (x,y) to (x, height)
  10. move the pen 20 pixels right along the x axis
  11. repeat steps 9 to 10 until (x) is equal to width

three

  1. create a canvas with the dimensions 800 x 600
  2. the canvas background is a light grey
  3. put the pen at coordinates 0,0
  4. set your initial (x,y) to (0,0)
  5. draw a 4px thick black line from coordinates (x,y) to (width,y)
  6. move the pen 20 pixels down along the y axis
  7. multiply the thickness of the line by .1 of current y coordinates
  8. repeat steps 5 to 6 until (y) is equal to (height)

four

  1. create a canvas with the dimensions 800 x 600
  2. draw 100 (10 x 10) rectangles in random positions within the confines of the canvas

five

  1. create a canvas with the dimensions 800 x 600
  2. draw 200 circles with 4px thick outline in random positions within the confines of the canvas
  3. for each cirlce, if the integer value of the x coordinate is divisible by 2, color the outline red, otherwise yellow

six

  1. create a canvas with the dimensions 800 x 600
  2. draw 200 circles with 4px thick outline in random positions within the confines of the canvas
  3. for each cirlce, if the integer value of the x coordinate is divisible by 2, color the outline red, otherwise yellow

seven

  1. create a canvas with the dimensions 720x360
  2. create a grid of 5x5 squares, spaced out evenly at 35px on the x and y axis

eight

  1. create a canvas with the dimensions 800x600
  2. draw a circle with a 4px solid black outline at the center of the canvas
  3. expand and detract the radius the circle based on the x position of the mouse

nine

  1. create a canvas with the dimensions 800x600
  2. set the background color to white
  3. draw a circle with a 4px solid black outline at the center of the canvas
  4. expand and detract the radius of the circle based on the x position of the mouse
  5. increase / decrease the green value of the background color based on the y position of the mouse

ten

  1. create a canvas with the dimensions 800x600
  2. set the background color to a dark grey
  3. put the pen at (0,0)
  4. draw a white 1px thick line from current coordinates to a coordinate with the same x and height of the canvas
  5. move the pen 2px to the right on the x axis
  6. repeat steps 4 to 5 until you reach the width of the canvas
  7. increase / decrease the distance between the lines based on the approximity of the mouse to the center of the canvas

eleven

  1. create a canvas with the dimensions 800x600
  2. set the background color to white
  3. put the pen at (0,0)
  4. draw a rectangle with the width of 20px from the coordinates (0,0) to (0,height) with the width of 4
  5. animate the rectangle along the x axis slowly
  6. when the rectangle reaches the far right of the canvas, and exits canvas, it enters from coordinates (0,0) again
  7. increase / decrease the speed of the moving rectangle based on the distance of the mouse position from the center of the canvas

twelve

  1. create a canvas with the dimensions 800x600
  2. set the background color to white
  3. click on any point on the screen
  4. draw a circle with yellow color and 4px thick black at the point of the click
  5. repeat steps 3 - 4 twenty times
  6. circles expand and detract based on a slow sine wave function
  7. based on the y coordinate of the mouse, the circles pulse faster or slower
  8. based ont he x coordinate of the mouse, the circles detract and expand in larger ranges

  • create a new file
  • paste the HTML below in the file

  • save your file as an HTML file (i.e: my-drawing.html) somewhere

  • with every change you can reload the browser and see the changes
